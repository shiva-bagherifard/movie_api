REST API Documentation

Movies Endpoints

Request URL Request Body Response
GET /movies N/A List of all movies
GET /movies/:title N/A Data about a single movie by title
GET /genres/:name N/A Data about a genre by name
GET /directors/:name N/A Data about a director by name

Users Endpoints

Request URL Request Body Response
POST /users/register { "Username": "example_user", "Password": "example_password", "Email": "user@example.com", "Birthday": "1990-01-01" } New user registration
PUT /users/:id { "Username": "updated_username", "Password": "updated_password", "Email": "updated_email@example.com", "Birthday": "1992-02-02" } Updated user info
POST /users/:id/favorites/add { "MovieId": "movie_id_to_add_to_favorites" } Added a movie to user's favorites
DELETE /users/:id/favorites/remove { "MovieId": "movie_id_to_remove_from_favorites" } Removed a movie from user's favorites
DELETE /users/:id/delete N/A Deregistered a user

CINE-VERSE EXPRESS API

Short description

The web application provides users with access to information about different
movies, directors, and genres. Users will be able to sign up, update their
personal information, and create a list of their favorite movies

Server

MOVIES: Managing movies API

GET
/movies lists all the movies
GET
/movies/{title} lists all data about a movie by the title
GET
/movies/genre/{genre} lists all the movie titles with a specific genre
GET
/movies/director/{directorName} lists data about a director
GET
/users lists all the users
POST
/users create a new user
PUT
/users/{id} update user name using the id
DELETE
/users/{id} remove user from the list
POST
/users/{id}/{movieTitle} add a movie to the user's favorite movies list
DELETE
/users/{id}/{movieTitle} delete a movie from the user's favorite movies list

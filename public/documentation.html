REST API Documentation

Movies Endpoints

Request URL Response
GET /movies List of all movies
GET /movies/:title Data about a single movie by title

Users Endpoints

Request URL Response
POST /users/register New user registration
PUT /users/:id Update user info
POST /users/:id/favorites/add Add a movie to user's favorites
DELETE /users/:id/favorites/remove Remove a movie from user's favorites
DELETE /users/:id/delete Deregister a user
